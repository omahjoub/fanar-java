# Contributing

Thank you for considering a contribution to the Fanar Java SDK. This guide covers standards, module creation, and
testing.

## Getting Started

1. Fork the repository and clone your fork.
2. Read [ARCHITECTURE.md](ARCHITECTURE.md) for the full design context.
3. Read [the ADRs](adr/INDEX.md) for the reasoning behind non-obvious decisions.
4. Create a feature branch from `main`.
5. Submit a pull request with a clear description of the change.
